Core OOP Principles

The Project Blueprint: Class Hierarchy

The entire system is built on a foundation of inheritance. A general, abstract `MediaItem` class provides a common structure for all specific media types, which then extend it with their own unique properties.

The Four Pillars of OOP

This project requires you to apply the fundamental principles of Object-Oriented Programming.

  • Abstraction

    `MediaItem` is abstract. It defines a contract (`displayDetails()`) without a concrete implementation, hiding complexity and focusing on essential features.

  • Inheritance

    `Book`, `Movie`, and `Audiobook` inherit properties and behaviors from `MediaItem`, promoting code reuse and establishing a clear hierarchy.

  • Polymorphism

    Calling `displayDetails()` on any `MediaItem` object results in a unique output specific to its actual type (`Book`, `Movie`, etc.). One action, many forms.

  • Encapsulation

    All properties are `protected`, bundling data with the methods that operate on it and restricting direct external access to ensure data integrity.

Polymorphism in Action

The `displayDetails()` method showcases polymorphism. Although called on a generic `MediaItem` reference, the specific implementation for each subclass is executed, resulting in varied output complexity.

Project Workflow: Tasks 1-5

Task 1: The Core Media Blueprint

Create the abstract `MediaItem` class to serve as the foundation.

Task 2: Creating Specialized Media Types

Define the `Book`, `Movie`, and `Audiobook` classes that extend `MediaItem`.

Task 3: Implementing Unique Behaviors

Provide a specific `displayDetails()` implementation for each concrete class.

Task 4: The Library Manager

Build the `Library` class to hold and manage a collection of `MediaItem` objects.

Task 5: Application Entry Point

Create the `Main` class to instantiate objects, populate the library, and run the program.

Evaluation for Success

Your submission is evaluated on four key areas. A strong project demonstrates mastery across all of them, showing not just a working program, but a well-designed and robust software solution.
